# 🍽️ MomoMia Café Website

A modern, responsive website built for a real local café — **MomoMia (Rampur Garden, Bareilly)**.
This project is designed to help small businesses establish an online presence and attract more customers.

---

## 🚀 Live Demo

👉 https://shraddha-1803.github.io/Local-Business-Website-FUTURE_FS_03/

---

## 📌 About the Project

This website was created as a real-world project to showcase how a simple, well-designed website can help a local café:

* Display menu items clearly
* Provide location & contact details
* Enable direct customer interaction via WhatsApp
* Improve visibility and professionalism

---

## ✨ Features

* 🎨 **Modern UI (Zomato-style cards)**
* 🌙 **Dark / Light Mode Toggle**
* 📱 **Fully Responsive (Mobile-Friendly)**
* 🍔 **Interactive Menu with Images**
* 📍 **Location Integration (Google Maps)**
* 💬 **WhatsApp “Order Now” Button**
* ✨ **Scroll Animations**
* 📌 **Sticky Navbar**

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🌐 Deployment

This project is deployed using **GitHub Pages**.

To deploy:

* Go to repository **Settings**
* Open **Pages**
* Select branch → `main`
* Save and get your live link
